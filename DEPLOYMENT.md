## Server Requirements

### FFmpeg Installation

FFmpeg must be installed on the server where the Streamlit app is running. Install using:

```bash
# On Ubuntu/Debian server
sudo apt update && sudo apt install ffmpeg

# On Amazon Linux/RHEL
sudo yum install ffmpeg

# On macOS server
brew install ffmpeg
```

Verify installation:
```bash
ffmpeg -version
```
